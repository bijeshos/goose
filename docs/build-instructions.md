# Build instructions

- clone the repository

- Run `go build -o go.examples github.com/bijeshos/goose`
    - to build the project

- Run `go install -o go.examples github.com/bijeshos/goose`
    - to install the binaries

- Run `./goose`
    - to run the binary

- Run `go run main.go`
    - to run main program
